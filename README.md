# 🏠 California House Price Prediction with MLP

Bu proje, California konut veri setini kullanarak ev fiyatlarını tahmin etmek için geliştirilmiş bir **Derin Öğrenme (Deep Learning)** çalışmasıdır.

## 🚀 Proje Özellikleri
- **Model:** Çok Katmanlı Algılayıcı (Multi-Layer Perceptron)
- **Kütüphaneler:** TensorFlow, Keras, Scikit-Learn, Seaborn, Matplotlib
- **Optimizasyon:** Adam Optimizer ve Dropout (%20) katmanı ile ezberleme (overfitting) engellenmiştir.
- **Metrikler:** R2 Score, MSE, MAE ve MAPE kullanılarak kapsamlı analiz yapılmıştır.

## 📊 Model Performansı
Modelimiz, test verileri üzerinde yaklaşık **%68 R2 skoru** elde etmiştir.
